currency.js

A small, lightweight javascript library for working with currency values. It resolves issues working with floating point numbers in javascript and includes a simple formatter to boot.

Methods

currency.add

Basic arithmetic.

currency(123.45).add(.01);    // 123.46
currency(987.65).add(123.45); // 1111.00

currency.subtract

currency(123.45).subtract(.01);    // 123.44
currency(123.45).subtract(987.65); // -864.20

currency.multiply

currency(123.45).multiply(2); // 246.90

currency.divide

currency(123.45).divide(2); // 61.73

currency.distribute

Distribute takes the amount and tries to distribute the it evenly. Any extra pennies left over from the distribution will be stacked onto the first set of entries.

currency(12.35).distribute(3); // [4.12, 4.12, 4.11]
currency(12.00).distribute(3); // [4.00, 4.00, 4.00]

currency.format

A basic formatter that outputs a friendly format.

currency(1000.00).format();              // "1,000.00"
currency("1,234,567.89").add("200,000"); // "1,434,567.89"

The formatter can be overridden by setting the currency.setting values.

currency.settings.seperator = ' ';
currency.settings.decimal = ',';

...

currency(1000.00).format();              // "1 000,00"
currency("1,234,567.89").add("200,000"); // "1 434 567,89"
