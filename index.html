Basic Usage

currency.js was built to resolve floating point issues in javascript and made to be easy to use.

When working with currencies, we only need the decimals to be precise up to the penny and don't want to deal with complex calculations every time we're trying to perform arithmetic with money. We resolve this issue by working entirely with integers behind the scenes, so you don't have to worry about decimal precision.

Examples:

2.51 + .01;                   // 2.5199999999999996
currency(2.51).add(.01);      // 2.52

2.52 - .01;                   // 2.5100000000000002
currency(2.52).subtract(.01); // 2.51

Also included is the ability to work with a variety of strings. This makes it easy to work into your user interface without having to conversions from string to numbers:

var c = currency("$1,234.56").add("890.12"); // 2124.68
c.format(); // 2,124.68

// Negative values
currency("-$5,000").add(1234.56);  // -3765.44
currency("($5,000)").add(1234.56); // -3765.44

By default, currency resolves to a string value. This makes it easy to perform calculations using inputs:

// Sets the first input to the resolved currency value
document.getElementsByTagName("input")[0].value = currency(1234.56).add(6.44); // 1241.00

currency.js has several basic arithmetic methods:

currency.add

currency(123.45).add(.01);    // 123.46
currency(987.65).add(123.45); // 1111.00

currency.subtract

currency(123.45).subtract(.01);    // 123.44
currency(123.45).subtract(987.65); // -864.20

currency.multiply

currency(123.45).multiply(2); // 246.90

currency.divide

currency(123.45).divide(2); // 61.73

currency.distribute

Distribute takes the amount and tries to distribute the it evenly. Any extra pennies left over from the distribution will be stacked onto the first set of entries.

currency(12.35).distribute(3); // [4.12, 4.12, 4.11]
currency(12.00).distribute(3); // [4.00, 4.00, 4.00]

currency.format

A basic formatter that outputs a friendly format.

currency(1000.00).format(); // "1,000.00"
currency("1,234,567.89").add("200,000").format(); // "1,434,567.89"

The formatter can be overridden by setting the currency.setting values.

currency.settings.seperator = ' ';
currency.settings.decimal = ',';

...

currency(1000.00).format(); // "1 000,00"
currency(1234567.89).add("200 000").format(); // "1 434 567,89"
